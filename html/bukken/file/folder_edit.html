フォルダ名を編集する
フォルダ名を編集します。
    フォルダツリーから編集する
  1. フォルダツリーから編集したいフォルダを選択し、 「編集」ボタンをクリックし、フォルダ名を編集します。
    2. サムネ表示
    サムネ表示
    サムネ表示
    注意 「/ : * ? ” < > | ¥」はフォルダ名に利用することが出来ません。 最上位のフォルダ「ファイル」は編集することが出来ません。
    一覧から編集する
  2. 編集したい項目の「」ボタンをクリックし、フォルダ名を編集します。

    一覧表示
